# Origami CV — Diagram to 3D Folding Instructions

A computer vision system that takes a photo of a flat origami diagram and outputs an animated 3D step-by-step folding guide with plain-English instructions.

## Stack
- YOLOv8 + Roboflow (symbol detection)
- Claude API (instruction generation)
- FastAPI + Docker + AWS (backend)
- Three.js + React + Vercel (frontend)
